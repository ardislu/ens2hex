ens2hex

About this site

What is this?

This is a tool to get the Ethereum hexadecimal (0x) address for every ENS (.eth) name in a list.

How does it work?

This tool generates a JSON-RPC request containing a payload to query the ENS PublicResolver smart contract for the hexadecimal addresses.

Then the tool sends the request to a public Ethereum node, which executes the payload and provides a response.

Credits

There are no dependencies other than the keccak256 implementation used to prepare the JSON-RPC payload. The rest of the tool consists of about 80 lines of vanilla JavaScript.

The keccak256 implementation was taken from Deno (original code here).
